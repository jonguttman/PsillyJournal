# Changelog

## [0.3.0] - 2026-02-23

### Added
- QR token system design document (docs/plans/2026-02-23-qr-token-system-design.md)
- Secure product verification via opaque tokens and server-authoritative proxy
- Structured routine tracker with schedules, reminders, and adherence tracking
- Offline pending queue for QR scans without connectivity
- Threat model with rate limiting and anti-enumeration protections
- App Store compliance checklist and App Review submission copy
- Full UI copy for scanning, linking, error, and offline states

- Rewrote FractalTexture as iridescent feather fractal with recursive fern/peacock patterns
- Color-shifting fronds: warm gold → rose → cool lavender (iridescent effect)
- Bilateral symmetry with fiddlehead-curling tips and glow halos
- Replaced scattered organic elements (mycelium, smoke wisps) with single elegant central fractal
